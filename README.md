# FAQ Web Page

## Overview

This project is a simple **Frequently Asked Questions (FAQ) webpage** built using **HTML and CSS only**.
It displays common customer questions and answers in a clean, readable layout that works on **desktop, tablet, and mobile devices**.

The page includes:

* A header section with the FAQ title
* FAQ questions and answers
* Ordered and unordered lists
* Responsive design for mobile devices
* Styled links and dividers

No frameworks or JavaScript are required.

---

To run this webpage you only need:

# How to Run the Webpage

### Step 1

Download or copy the project folder.

### Step 2

Open the folder.

### Step 3

Locate the file:

```
index.html
```

### Step 4

Double-click the file.

2. Drag and drop the **index.html** file into the browser window.

# How to Edit the Content

You can modify the FAQ questions and answers by editing the **index.html** file.

Open the file in a code editor such as:

* VS Code
* Sublime Text
* Atom
* Notepad++

Example FAQ section:

```html
<h2>How can I track my order?</h2>

<p>You can track your order using the following steps:</p>

<ol>
<li>Go to the Order Tracking page.</li>
<li>Enter your order number and email address.</li>
<li>Click Track Order.</li>
</ol>
```

You can change the text inside these tags to add your own questions and answers.

---

# Customizing the Design

All styles are written inside the `<style>` section in the `<head>` of the HTML file.

Example:

```css
.header{
    background:#f1f1f3;
    text-align:center;
    padding:60px 20px;
}
```

You can modify colors, fonts, spacing, and layout here.
